- `"use client"`, import `useTambo` and `useTamboThreadList` from `@tambo-ai/react`.
- Outer div: `className="retro-panel"` with `style={{ width: 260, flexShrink: 0, borderRight: "1px solid #ffb000", display: "flex", flexDirection: "column", height: "100%", overflow: "hidden" }}`.

- Heading: `<h2 className="retro-text" style={{ fontSize: 16, margin: "0 0 12px 0" }}>> THREADS</h2>`.
- New Thread button: `<button className="retro-btn retro-btn-primary" style={{ width: "100%" }} onClick={() => startNewThread()}>[+] NEW THREAD</button>`.
- Thread list: `<div style={{ overflowY: "auto", flex: 1 }}>`. Map threads to divs with `className={"retro-thread-item" + (thread.id === currentThreadId ? " active" : "")}`. Display `> ${thread.name ?? thread.id.slice(0, 8)}` (with > prefix for terminal feel).

**RetroMessageHistory.tsx:**
- `"use client"`, import `useRef, useEffect` from React, import `useTambo` from `@tambo-ai/react`.
- Auto-scroll with `bottomRef` pattern (same as Primer).
- Outer div: `style={{ flex: 1, overflowY: "auto", padding: 16, background: "#050505" }}`.
- Map over `messages`. Extract text same way as Primer.
- User messages: wrapped in div with `style={{ display: "flex", justifyContent: "flex-end", marginBottom: 12 }}`, inner div `className="retro-bubble-user"`.
- AI messages: wrapped in div with `style={{ display: "flex", justifyContent: "flex-start", marginBottom: 12 }}`, inner div `className="retro-bubble-ai"`.
- Loading state (isStreaming || isWaiting): `<div className="retro-loading" style={{ padding: "8px 0" }}>Processing query...</div>`.

**RetroMessageInput.tsx:**
- `"use client"`, import `useTamboThreadInput` from `@tambo-ai/react`.
- Outer div: `style={{ flexShrink: 0, padding: 12, borderTop: "1px solid #ffb000", display: "flex", gap: 8, alignItems: "flex-end", background: "#050505" }}`.
- Textarea: `<textarea className="retro-input" rows={2} placeholder="> Enter command... (Ctrl+Enter to transmit)" value={value} onChange={e => setValue(e.target.value)} onKeyDown={handleKeyDown} disabled={isDisabled} />` wrapped in `<div style={{ flex: 1 }}>`.
- Send button: `<button className="retro-btn retro-btn-primary" disabled={isDisabled} onClick={() => submit()}>TRANSMIT</button>`.
- `handleKeyDown`: same Ctrl/Meta+Enter pattern as Primer.
