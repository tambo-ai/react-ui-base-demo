  <action>
Follow the Primer reference pattern exactly. All three components are "use client", import Tambo hooks, use plain HTML elements with nes.css class names for styling, and inline styles for layout positioning.

**NesSidebar.tsx:**
- `"use client"`, import `useTambo` and `useTamboThreadList` from `@tambo-ai/react`.
- Outer div: `style={{ width: 260, flexShrink: 0, borderRight: "4px solid #fff", display: "flex", flexDirection: "column", height: "100%" }}`.
- Header area: `className="nes-container with-title"` with `<p className="title">Threads</p>`.
- New Thread button: `<button className="nes-btn is-primary" onClick={() => startNewThread()}>+ New Thread</button>`.
- Thread list: `<ul className="nes-list is-disc">` mapping over `threads`. Active thread gets `className="nes-text is-primary"`. Each li has `onClick={() => switchThread(thread.id)}`, `style={{ cursor: "pointer", marginBottom: 8 }}`. Display `thread.name ?? thread.id.slice(0, 8)`.
- While `isLoading`, show nothing (same as Primer pattern).

**NesMessageHistory.tsx:**
- `"use client"`, import `useRef, useEffect` from React, import `useTambo` from `@tambo-ai/react`.
- Auto-scroll with `bottomRef` pattern (same as Primer).
- Outer div: `style={{ flex: 1, overflowY: "auto", padding: 16 }}`.
- Map over `messages`. Extract text the same way as Primer: `msg.content.filter(c => c.type === "text").map(c => "text" in c ? c.text : "").join("")`.
- User messages: `<div className="nes-balloon from-right">` with `style={{ marginBottom: 12, textAlign: "right" }}`.
- AI messages: `<div className="nes-balloon from-left">` with `style={{ marginBottom: 12 }}`.
- Loading state (isStreaming || isWaiting): `<progress className="nes-progress is-primary" value="70" max="100" style={{ height: 20 }} />`.
- Bottom scroll anchor div.

**NesMessageInput.tsx:**
- `"use client"`, import `useTamboThreadInput` from `@tambo-ai/react`.
- Outer div: `style={{ flexShrink: 0, padding: 12, borderTop: "4px solid #fff", display: "flex", gap: 8, alignItems: "flex-end" }}`.
- Textarea: `<textarea className="nes-textarea" rows={2} placeholder="Type a message... (Ctrl+Enter to send)" value={value} onChange={e => setValue(e.target.value)} onKeyDown={handleKeyDown} disabled={isDisabled} style={{ flex: 1 }} />`.
- Send button: `<button className="nes-btn is-success" disabled={isDisabled} onClick={() => submit()}>Send</button>`.
- `handleKeyDown`: same Ctrl/Meta+Enter pattern as Primer.
