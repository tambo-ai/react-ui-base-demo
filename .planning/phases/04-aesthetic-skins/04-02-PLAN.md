- `"use client"`, import `useTambo` and `useTamboThreadList` from `@tambo-ai/react`.
- Outer div: `style={{ width: 260, flexShrink: 0, borderRight: "3px solid #000", display: "flex", flexDirection: "column", height: "100%", background: "#FFF8DC", padding: 16 }}`.
- Heading: `<h2 style={{ fontWeight: 800, fontSize: 18, marginBottom: 12, fontFamily: "system-ui, sans-serif" }}>Threads</h2>`.
- New Thread button: `<button className="neo-btn neo-btn-primary" style={{ width: "100%", marginBottom: 16 }} onClick={() => startNewThread()}>+ New Thread</button>`.
- Thread list: div with `style={{ display: "flex", flexDirection: "column", gap: 6, overflowY: "auto", flex: 1 }}`. Map threads to divs with `className={"neo-thread-item" + (thread.id === currentThreadId ? " active" : "")}`. Display `thread.name ?? thread.id.slice(0, 8)`.

**NeoMessageHistory.tsx:**
- `"use client"`, import `useRef, useEffect` from React, import `useTambo` from `@tambo-ai/react`.
- Auto-scroll with `bottomRef` pattern (same as Primer).
- Outer div: `style={{ flex: 1, overflowY: "auto", padding: 16, background: "#FAFAF9" }}`.
- Map over `messages`. Extract text same way as Primer.
- User messages: wrapped in div with `style={{ display: "flex", justifyContent: "flex-end", marginBottom: 12 }}`, inner div `className="neo-bubble-user"`.
- AI messages: wrapped in div with `style={{ display: "flex", justifyContent: "flex-start", marginBottom: 12 }}`, inner div `className="neo-bubble-ai"`.
- Loading state: a div with `className="neo-container"` styled as a small loading indicator: `style={{ padding: "8px 16px", display: "inline-block" }}` containing "Thinking..." in bold text.

**NeoMessageInput.tsx:**
- `"use client"`, import `useTamboThreadInput` from `@tambo-ai/react`.
- Outer div: `style={{ flexShrink: 0, padding: 12, borderTop: "3px solid #000", display: "flex", gap: 8, alignItems: "flex-end", background: "#FAFAF9" }}`.
- Textarea: `<textarea className="neo-textarea" rows={2} placeholder="Type a message... (Ctrl+Enter to send)" value={value} onChange={e => setValue(e.target.value)} onKeyDown={handleKeyDown} disabled={isDisabled} />` wrapped in `<div style={{ flex: 1 }}>`.
- Send button: `<button className="neo-btn neo-btn-success" disabled={isDisabled} onClick={() => submit()}>Send</button>`.
- `handleKeyDown`: same Ctrl/Meta+Enter pattern as Primer.
